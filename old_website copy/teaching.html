MOOC: Begin Programming (2013 - 2014)

The University ran their first online course titled "Begin programming: build your first mobile game" with 10,000 participants on the first run and over 30,000 on the second run. The course involved teaching the participants how to learn programming from scratch by teaching them how to program a game for Android-based phones using the Android SDK with Java. This position involved helping users with their problems such as errors installing the Android SDK and further explaining them programming principals when they couldn't grasp the concepts from the provided video.
